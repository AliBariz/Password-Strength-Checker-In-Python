# 🔒 Password Strength Checker

An interactive **password strength checker** built with **Python and Streamlit**.  
This app helps you create strong and secure passwords by evaluating them against best practices and giving suggestions to improve them.

---

## 🕹️ How It Works

The app checks your password based on:

1. Minimum length (8 characters or more)
2. Inclusion of **uppercase letters**
3. Inclusion of **lowercase letters**
4. Inclusion of **numbers**
5. Inclusion of **special characters** (e.g., !@#$%^&*)

It then provides a **strength rating**: Weak ⚠️, Moderate 😐, Strong 👍, or Very Strong 💪, along with tips to improve your password.
